# Rockwell-B-1-Lancer
Rockwell B-1 Lancer aircraft mod for ArmA3
# Info
The Rockwell B-1 Lancer is a four-engine supersonic variable-sweep wing, jet-powered heavy strategic bomber used by the United States Air Force.
